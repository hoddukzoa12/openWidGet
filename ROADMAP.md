# OpenWidGet Roadmap

## v0.1-alpha — public proof-of-work

Goal: prove that OpenWidGet is a real, runnable Windows desktop app shell and a credible open-source starting point.

Must-have:

- Tauri desktop app shell
- Basic tray menu
- Bundled widget preview surface
- English-primary README and separated Korean docs
- Windows manual smoke checklist
- Unsigned Windows `.exe` installer or documented blocker
- Portable zip release candidate
- Demo media and launch copy for X/LinkedIn

## Post-alpha

- Real floating overlay widget runtime
- Anchor Shortcut / Live Shortcut Group lifecycle
- Widget position persistence
- `widget.json` schema and validator
- Built-in Clock, Memo, Project Launcher, and GitHub widgets
- Widget gallery and registry admission workflow
- Safer permissions and sandbox model

## Not in v0.1-alpha

- Marketplace
- Cloud sync
- Code signing
- Full security sandbox
- Full multi-monitor support
- Production-ready installer UX
