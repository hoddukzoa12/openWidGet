# OpenWidGet 프로젝트 노트

## 문제의식

Windows 데스크톱 위젯은 여전히 폐쇄적이거나 커스터마이징 장벽이 높습니다. OpenWidGet은 위젯을 작은 웹 앱처럼 만들고 공유할 수 있는 오픈소스 런타임을 실험합니다.

## v0.1-alpha 목표

- 실행 가능한 Tauri 앱 스켈레톤
- 샘플 위젯 프리뷰
- Windows 직접 테스트 루프
- `.exe`/portable 릴리즈 준비
- 영어 중심 OSS 문서와 분리된 한국어 문서

## 공개 포지셔닝

이 프로젝트는 완성형 플랫폼이 아니라 public proof-of-work입니다. X/LinkedIn에는 “오픈소스 위젯 런타임을 만들기 시작했다”는 정직한 톤으로 공개합니다.

## 테스트 원칙

먼저 만들고, 의미 있는 체크포인트마다 실제 Windows에서 직접 실행해 검증합니다. 단, 공개 릴리즈와 데모 영상은 반드시 Windows 실기 증거 이후에 진행합니다.
